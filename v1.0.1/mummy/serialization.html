mummy.serialization.dumps()¶

serialize a native python object into an mummy string

Parameters:
  • object – the python object to serialize
  • default (function) – If the ‘object’ parameter is not serializable and this parameter is provided, this function will be used to generate a fallback value to serialize. It should take one argument (the original object), and return something serializable.
  • compress (bool) – whether or not to attempt to compress the serialized data (default True)
Returns:

the bytestring of the serialized data
